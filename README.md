Tetris
======
Unity 2D Game Tetris

### 1. Unity Version
Version 4.2


### 2. How to Run Demo
Web Player Demo:<br/>
Open the page by your browser.<br/>
        
        test/unity/Tetris/Tetris.html
        

### 3. How to import into your Unity project.
In Unity, File -> Open Project -> Open Other -> Select to the Tetris
